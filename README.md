# ダンゴムシのミニツールボックス

シンプルで使いやすい、ブラウザで動くミニツールとミニゲームのコレクションです。

## 公開URL
https://dangomushi2357.github.io/tool-box/index.html

## 収録ツール

### 数取機 (Counter)
シンプルながらも「使い心地」にこだわったカウンターです。
- **こだわり**: ボタンを押した際の立体的な「ななめ沈み込み」アクション。
- **インテリジェンス**: 数字がマイナスになると自動的に文字が赤くなり、直感的な視認性を高めています。
- **保存機能**: ブラウザの `localStorage` を使用し、ページを閉じてもカウントを保持します。

### クリックゲーム (Click Game)
連打の限界に挑むシンプルなゲームです。
時々、何か仕掛けがあるかも？？
- **デザイン**: 押し心地の良いエメラルドグリーンのボタンデザインを採用。
- **レスポンシブ**: PCでもスマホでも快適に操作可能です。
